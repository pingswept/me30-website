## Sensor amplification

Sensors produce real small voltages, so we need to amplify them.

{{< katex display >}}
V_{in} = V_{out} * \frac{R_1}{(R_1 + R_2)}
{{< /katex >}}

{{< katex display >}}
\frac{V_{in}}{V_{out}} = \frac{R_1}{(R_1 + R_2)}
{{< /katex >}}

{{< katex display >}}
Gain = \frac{V_{out}}{V_{in}} = \frac{(R_1 + R_2)}{R_1}
{{< /katex >}}

{{< katex display >}}
Gain = 1 + \frac{R_2}{R_1}
{{< /katex >}}
